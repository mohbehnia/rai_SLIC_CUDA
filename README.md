# SLIC_CUDA
Superpixel Segmentation SLIC of Achanta et al. [PAMI 2012, vol. 34, num. 11, pp. 2274-2282]. GPU version with CUDA.

### Library required :
* opencv 3.0 min
* cuda compute capability 3.0 min

### Example
  Cuda : 30 fps
